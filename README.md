# Conservation Genetics & Landscape Sustainability

This is the GitHub repository used to manage materials for the course *Conservation genetics and Landscape sustainability*. You likely meant to navigate towards the rendered html via the website [here](https://MWGCLS.github.io/Conservation-Genetics-Landscape-Sustainability).

The website can be rendered locally with `jekyll` after cloning the repo. For example:
```
git clone https://github.com/MWCGLS/MWCGLS.github.io.git
cd MWCGLS.github.io
bundle install
bundle exec jekyll serve --watch
```

For stability, this project pins `github-pages` in the Gemfile to a tested release. When you intentionally upgrade it, run `bundle update github-pages` and verify the site with a local build.
